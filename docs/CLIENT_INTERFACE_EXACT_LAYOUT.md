## 4. 关键实现细节

### 4.1 模式切换

**切换方式**：
- 点击"前进模式"按钮 → 切换到前进模式
- 点击"倒车模式"按钮 → 切换到倒车模式
- 默认模式：前进模式

**切换时变化**：
- 顶部标签："前进模式" / "倒车模式"
- 左侧视图标签："左侧视图" / "左侧视野"
- 中间视图标签："前方摄像头" / 无标签
- 右侧地图标签："高精地图" / "前方视图"
- 右侧视图：显示 / 隐藏（倒车模式显示"后方视图"）
- 控制按钮网格：12个 / 15个
- 方向盘可视化：显示 / 隐藏
- 档位：D / R
- 时间显示：温度 / 日期

---

### 4.2 按钮ToolTip

**实现方式**：
```qml
Button {
    ToolTip.visible: hovered
    ToolTip.text: modelData.name  // 中文名称
    ToolTip.delay: 300
    
    contentItem: Text {
        text: modelData.icon  // 只显示图标
    }
}
```

**效果**：
- 光标悬停：显示中文名称ToolTip
- 光标离开：ToolTip自动消失
- 延迟300ms显示（避免误触发）

---

### 4.3 方向盘可视化

**位置**：底部控制面板，中右区域（状态和清扫进度上方）

**显示条件**：仅在前进模式显示

**功能**：
- 方向盘图形：圆形，3条辐条，12个把手点
- 实时旋转：随滑块角度旋转（-180° 到 180°）
- 控制滑块：范围 -100 到 100
- 角度显示：实时显示当前角度
- MQTT同步：自动发送控制命令
