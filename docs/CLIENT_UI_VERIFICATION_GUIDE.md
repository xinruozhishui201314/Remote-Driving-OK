# 客户端 UI 完整验证指南

## Executive Summary

本文档提供远程驾驶客户端UI界面的完整验证指南，包括登录界面、车辆选择界面和远程驾驶操作界面的详细验证步骤。

**验证目标**：确保UI界面符合设计规范（`client/picture/远程驾驶客户端UI设计.png`），功能流程完整，界面美观。

---

## 1. 验证前准备

### 1.1 环境检查

```bash
# 1. 检查服务状态
docker compose ps | grep client-dev

# 2. 清除登录状态（确保从登录界面开始）
bash scripts/clear-client-login.sh

# 3. 编译客户端
docker compose exec client-dev bash -c "cd /tmp/client-build && make -j4"

# 4. 检查设计图片
file client/picture/远程驾驶客户端UI设计.png
```

### 1.2 X11 配置（如需要）

```bash
# 允许 Docker 容器访问 X11
xhost +local:docker

# 设置 DISPLAY 环境变量
export DISPLAY=:0
```
