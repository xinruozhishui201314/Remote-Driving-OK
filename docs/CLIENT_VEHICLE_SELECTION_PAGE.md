# 车辆选择页面优化

## 概述

将车辆选择对话框改为全屏嵌入页面，采用网格卡片布局，更适合显示多个车辆（特别是环卫扫地车）。

## 主要改动

### 1. 创建新的车辆选择页面

**文件**：`client/qml/VehicleSelectionPage.qml`

**特点**：
- ✅ 全屏嵌入页面（不再是弹窗）
- ✅ 网格布局（GridView，3列）
- ✅ 卡片式车辆展示
- ✅ 适合显示多个车辆

### 2. 页面布局

#### 顶部区域
- **标题**："选择车辆"（36px，加粗）
- **副标题**："请选择要控制的环卫扫地车"（16px，灰色）
- **装饰线**：渐变下划线
- **刷新按钮**：右上角

#### 中间区域（车辆网格）
- **布局**：GridView，3列网格
- **卡片尺寸**：每张卡片约 330x220px
- **卡片内容**：
  - 车辆图标区域（100px 高度，带选中指示器）
  - 车辆名称（18px，加粗）
  - VIN 信息（12px，灰色）
  - 选择按钮

#### 底部区域（操作栏）
- **当前选择车辆**：显示选中的车辆名称
- **错误信息**：红色背景框
- **会话信息**：如果已创建会话，显示会话信息
- **操作按钮**：
  - 创建会话（蓝色渐变）
  - 确认并进入驾驶（绿色渐变）

### 3. 车辆卡片设计

**视觉元素**：
- 深色背景（`#1A1A2A`）
- 选中状态：蓝色边框（`#4A90E2`，3px）+ 光晕效果
- 悬停效果：背景色变化
- 圆角：16px
- 阴影效果

**车辆图标区域**：
- 渐变背景
- 车辆图标（🚛 emoji，50px）
- 选中指示器（右上角圆形，带 ✓）

### 4. 交互优化

- ✅ 点击卡片或按钮都可以选择车辆
- ✅ 选中状态有明显的视觉反馈
- ✅ 悬停效果
- ✅ 页面进入动画（透明度 + 缩放）

### 5. 主窗口集成

**修改**：`client/qml/main.qml`

- 添加 `showVehicleSelection` 属性控制页面显示
- 登录成功后自动显示车辆选择页面
- 确认后进入主驾驶界面

## 布局特点

### 网格布局优势

1. **适合多车辆**：网格布局可以同时显示多个车辆，一目了然
2. **空间利用**：充分利用屏幕空间
3. **视觉清晰**：每个车辆独立卡片，信息层次清晰
4. **易于扩展**：可以轻松添加更多车辆信息（如状态、位置等）

### 卡片式设计优势

1. **信息集中**：每个车辆的所有信息集中在一个卡片中
2. **操作直观**：选择按钮就在卡片上，操作方便
3. **状态明确**：选中状态一目了然

## 使用流程

1. **登录** → 显示登录页面
2. **登录成功** → 自动显示车辆选择页面
3. **选择车辆** → 点击车辆卡片或按钮
4. **创建会话**（可选）→ 点击"创建会话"按钮
5. **确认进入** → 点击"确认并进入驾驶"按钮
6. **主界面** → 显示视频和控制面板
