## 启动客户端

### 快速启动命令

```bash
bash scripts/run-client-ui.sh
```

### 手动启动命令

```bash
# 1. 设置 X11 权限
xhost +local:docker

# 2. 运行客户端
docker compose exec -e DISPLAY=$DISPLAY client-dev bash -c 'cd /tmp/client-build && ./RemoteDrivingClient'
```
