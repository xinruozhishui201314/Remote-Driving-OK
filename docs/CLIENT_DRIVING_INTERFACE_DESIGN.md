# 远程驾驶界面设计实现

## Executive Summary

根据设计图片（`client/picture/远程驾驶客户端UI设计.png`）实现了完整的远程驾驶界面，包含前进模式、倒车模式、多视频视图、车辆状态显示、控制按钮等所有组件，所有组件均使用中文标签。

**实现文件**：`client/qml/DrivingInterface.qml`

---

## 1. 界面布局结构

### 1.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│  状态栏（StatusBar）                                      │
├─────────────────────────────────────────────────────────┤
│  [前进模式] [倒车模式]                   时间  温度      │
├──────────┬──────────────────────────┬───────────────────┤
│          │                          │                   │
│  左侧    │      前方摄像头/          │   导航地图        │
│  视图    │      倒车模式             │   (后方视图)      │
│          │                          │                   │
│          │                          │                   │
│  右侧    │                          │   系统警报        │
│  视图/   │      仪表盘区域           │   - 系统警报 1    │
│  PIP     │   ┌─────┬─────┬─────┐   │   - 系统警报 2    │
│          │   │水箱 │速度 │状态 │   │                   │
│          │   │水位 │档位 │清扫 │   │                   │
│          │   └─────┴─────┴─────┘   │                   │
│          │                          │                   │
│  控制    │                          │                   │
│  按钮    │                          │                   │
│  (12个)  │                          │                   │
└──────────┴──────────────────────────┴───────────────────┘
```

### 1.2 布局比例

- **左侧面板**：25% 宽度
  - 左侧视图（大）
  - 右侧视图/PIP（大）
  - 控制按钮网格（12个按钮，4列3行）

- **中央面板**：50% 宽度
  - 主视频视图（前方摄像头/倒车模式，65%高度）
  - 仪表盘区域（35%高度）
    - 左侧：水箱水位、垃圾箱填充
    - 中央：速度表、档位
    - 右侧：左侧状态、右侧状态、清扫进度

- **右侧面板**：25% 宽度
  - 顶部工具栏（亮度、警告、音频、设置）
  - 导航地图（50%高度）
  - 系统警报（50%高度）

## 3. 模式切换逻辑

### 3.1 前进模式

**切换方式**：点击"前进模式"按钮

**界面变化**：
- 主视频视图标签：`前方摄像头`
- 左侧面板底部视图标签：`右侧视图`
- 档位自动切换为：`D`（蓝色）
- 主视频视图不显示网格线

---

### 3.2 倒车模式

**切换方式**：点击"倒车模式"按钮

**界面变化**：
- 主视频视图标签：`倒车模式`
- 左侧面板底部视图标签：`PIP`
- 档位自动切换为：`R`（红色）
- 主视频视图显示红色网格线（停车辅助）
- PIP视图显示红色网格线

## 4. 数据绑定

### 4.1 车辆状态数据

**数据源**：`vehicleStatus`（VehicleStatus对象）

**绑定属性**：
- `vehicleSpeed`：从 `vehicleStatus.speed` 获取
- `currentGear`：根据 `forwardMode` 自动切换（D/R）
- `waterTankLevel`：模拟数据（75%），后续从车辆数据获取
- `trashBinLevel`：模拟数据（40%），后续从车辆数据获取
- `cleaningProgress`：模拟数据（80%），后续从车辆数据获取
- `cleaningCurrent`：模拟数据（400），后续从车辆数据获取
- `cleaningTotal`：模拟数据（500），后续从车辆数据获取

## 5. 样式规范

### 5.1 颜色方案

- **主色调**：蓝色（#4A90E2）
- **成功/正常**：绿色（#50C878）
- **警告/错误**：红色（#FF6B6B）
- **警告**：黄色（#FFAA00）
- **背景色**：深色（#0F0F1A, #1A1A2A）
- **文字色**：白色（#FFFFFF）、浅灰色（#B0B0B0）

### 5.2 字体

- **中文字体**：WenQuanYi Zen Hei、WenQuanYi Micro Hei、Noto Sans CJK SC
- **标题**：14-16px，粗体
- **数值**：20-48px，粗体
- **普通文字**：12-14px

### 5.3 圆角

- **卡片**：8px
- **按钮**：4-6px
- **进度条**：4px
