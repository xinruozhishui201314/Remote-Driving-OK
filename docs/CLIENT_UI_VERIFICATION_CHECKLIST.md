## 11. 图标与资源（组件显示不一致时替换用）

若某组件与设计图不一致，可替换为从网上下载的 SVG 或从设计图裁剪的控件图片，再按本表核对。

| 区域 | 组件 | 当前资源路径 | 说明 |
|------|------|--------------|------|
| 顶栏左侧 | 近光/远光/雾灯/雨刷/大灯/警示 | `icon/light.svg`、`highbeam.svg`、`foglight.svg`、`wiper.svg`、`light.svg`、`warning.svg` | 激活态由 QML 绿框/红框控制；图标可替换为设计图同款 SVG |
| 顶栏右侧 | 太阳 | `icon/sun.svg` | 新增，与设计图「太阳」对应 |
| 顶栏右侧 | 警告 | `icon/warning.svg` | 可替换为三角叹号等 |
| 顶栏右侧 | 齿轮 | `icon/settings.svg` | 可替换为齿轮形 SVG |
| 顶栏右侧 | 位置 | `icon/location.svg` | 可替换为图钉形 SVG |
| 顶栏右侧 | 云 | `icon/weather.svg` | 可替换为云形 SVG |
| 高精地图 | 缩放 | `icon/zoom.svg` | 放大镜 SVG，已替换原「⊕」文字 |

**替换步骤：**
1. 从 [Feather Icons](https://feathericons.com/)、[Heroicons](https://heroicons.com/) 等下载对应 SVG，或从设计图裁剪该控件为 PNG/SVG。
2. 覆盖 `client/qml/icon/` 下对应文件（如 `zoom.svg`、`sun.svg`），或新增文件并在 `DrivingInterface.qml` 中修改 `source` 路径。
3. 运行 `bash scripts/run-client-ui.sh --reset-login` 进入主驾驶界面，逐项与设计图对比。
4. 将核对结果更新到本清单对应行的「当前实现」与「状态」列。
