## 🎯 核心要点（必读）

### **一句话根本原因**
> 环境变量 `CLIENT_MEDIA_REQUIRE_HARDWARE_DECODE=1` 强制硬解，但容器中 VAAPI 初始化失败，代码拒绝软解 → 黑屏。

### **立即修复（< 1 分钟）**
```bash
bash fix_video_black_screen.sh
```

### **修复原理**
```
清空环境变量 或 修改配置 media.require_hardware_decode: false
  ↓
allowSoftDecode() 返回 true
  ↓
解码器使用 FFmpegSoftDecoder (CPU)
  ↓
视频正常显示
```
